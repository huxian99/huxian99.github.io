Android 当用户停用app之后

今天遇到一个问题，测试妹子想用微信登录app，但是点击微信的小图标却怎么也没有反应，于是提了bug给我。我马上查了查相应的代码，大致的逻辑是根据sdk提供的isWXAppInstalled()方法来做判断，如果用户安装了微信就跳转微信登录界面，如果用户没有安装就给一个提示。这明明没什么问题啊，于是找到了出问题的三星手机。发现原来是isWXAppInstalled()方法返回了true，导致点击后没有任何反应。测试妹子跟我说这台s4预装了微信，但被她停用了，会不会是这个问题！后来，后来就有了如下的代码:

try {
   PackageManager manager = activity.getPackageManager();
   PackageInfo packageInfo = manager.getPackageInfo("com.tencent.mm", 0); // 1
   ApplicationInfo applicationInfo = packageInfo.applicationInfo;  
   boolean enabled = applicationInfo.enabled;  // 2
   if (!enabled) {
      Toast.makeText(activity, "balabala", Toast.LENGTH_SHORT).show();
      return;
   }
} catch (PackageManager.NameNotFoundException e) {
   e.printStackTrace();
}

1) 先通过adb命令找出微信的packageName获取相应的PackageInfo
2) 通过applicationInfo.enabled来判断应用是否可用
